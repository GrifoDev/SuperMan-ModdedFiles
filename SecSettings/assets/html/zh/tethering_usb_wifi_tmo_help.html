USB网络分享

使用 USB 网络共享以通过 USB 线缆与单一 PC 共享设备的移动网络连接。若要打开 USB 网络共享，您必须将智能手机移动热点服务添加至您的费率方案。
若要网络共享您的设备，请使用 USB 线缆将其连接至电脑。然后，在您的移动设备上，进入设定 > 连接 > 移动热点和网络共享，并选择 USB 网络共享。
USB 网络共享适用于 Windows 和 Linux。
若需更多信息，包括如何将 USB 网络共享用于其他操作系统，请进入 http://www.android.com/tether。
对于 Windows XP，您可能必须在 PC 上安装 USB 驱动程序，才能网络共享您的设备。若要安装驱动程序，进入 http://www.samsung.com/support/。
当您使用 USB 数据线将设备连接至 PC 时，您不仅可以通过网络共享来共享移动网络连接，还可以共享文件。您无法同时共享两者。如果您想要使用 PC 访问您设备的 SD 卡，请勿启用 USB 网络共享。

移动热点

将您的设备变为移动热点，以便与其他设备共享您的移动数据连接。
若要打开移动热点，您必须将智能手机移动热点服务添加至您的费率方案。
若要打开智能手机移动热点，请进入设定 > 连接 > 移动热点和网络共享，然后选择移动热点。
若要配置智能手机移动热点，轻敲移动热点，设置移动热点。
若要设定有多少设备可使用您的移动网络连接，设置移动热点 > 显示高级选项 > 最大连接数。
