Tethering USB

Utilizzate il tethering USB per condividere la connessione di rete mobile del dispositivo con un solo PC con un cavo USB. Per attivare il tethering USB, è necessario aggiungere il servizio Smartphone Router Wi-Fi al piano tariffario.
Per eseguire il tethering del dispositivo, collegatelo a un computer con un cavo USB, quindi dal dispositivo mobile, andate in Impostazioni > Connessioni > Router Wi-Fi e tethering e selezionate Tethering USB.
Il tethering USB funziona con Windows e Linux
Per ulteriori informazioni, incluse quelle relative all'utilizzo del tethering USB con altri sistemi operativi, andate all'indirizzo http://www.android.com/tether.
Su Windows XP, prima di poter utilizzare il tethering del dispositivo, potrebbe essere necessario installare un driver USB sul PC. Per installare i driver, andate all'indirizzo http://www.samsung.com/support/.
Connettendo il dispositivo a un PC con un cavo USB, potrete condividere dei file o la connessione dati di rete mobile tramite tethering. Non è possibile fare entrambe le cose. Non attivate il tethering USB se desiderate utilizzare il PC per accedere alla scheda SD del dispositivo.

Router Wi-Fi

Convertite il dispositivo in un Router Wi-Fi in modo da condividere la connessione dati mobile con altri dispositivi.
Per attivare Router Wi-Fi è necessario aggiungere il servizio Smartphone Router Wi-Fi al proprio piano tariffario.
Per attivare Smartphone Router Wi-Fi, andate in Impostazioni > Connessioni > Router Wi-Fi e tethering e selezionate Router Wi-Fi.
Per configurare Smartphone Mobile Hotspot, toccate Router Wi-Fi, quindi Configura Router Wi-Fi.
Per impostare il numero di dispositivi che possono utilizzare la connessione di rete mobile, andate in Configura Router Wi-Fi > Mostra opzioni avanzate > Connessioni massime.
