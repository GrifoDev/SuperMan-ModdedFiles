Tethering USB

È possibile utilizzare il tethering per condividere la connessione dati mobile del proprio telefono con un computer tramite un cavo USB.
Per eseguire il tethering, collegare il telefono a un computer con un cavo USB, quindi toccare sul telefono per selezionare “Tethering USB” in “Impostazioni >> Connessioni >> Router Wi-Fi e tethering”.
Il tethering funziona con Windows®7, Windows Vista® e Linux. Per ulteriori informazioni riguardo al collegamento tramite tethering con altri sistemi operativi, visitare il sito Web www.android.com/tether.
Su Windows XP, prima di poter utilizzare il tethering tramite il proprio telefono potrebbe essere necessario installare un driver USB sul computer. Per installare i driver, visitare il seguente URL sul browser del computer.
<http://www.samsung.com/support/>
Connettendo il telefono a un computer con un cavo USB, è possibile condividerne la connessione dati mobile OPPURE condividere file. Non è possibile fare entrambe le cose. Non attivare il tethering se si desidera utilizzare il computer per accedere alla scheda di memoria del telefono.

Router Wi-Fi

Convertire il telefono in punto di accesso Wi-Fi in modo da condividere la connessione dati mobile con altri dispositivi.
Per attivare Smartphone Mobile Hotspot, toccare per selezionare “Router Wi-Fi” in “Impostazioni >> Connessioni >> Router Wi-Fi e tethering”.
Per configurare Smartphone Mobile Hotspot, toccare “Router Wi-Fi” quindi “Configura Router Wi-Fi”.
È possibile impostare quanti dispositivi possono utilizzare la connessione dati mobile in “Configura Router Wi-Fi >> Mostra opzioni avanzate >> Connessioni massime”.
