USB 테더링

USB 케이블을 통해 휴대폰의 모바일 데이터 연결을 단일 컴퓨터와 공유하려면 테더링을 사용하세요.
휴대폰의 테더링 기능을 사용하려면 사용하는 요금제에 스마트폰 모바일 핫스팟 서비스를 추가해야 합니다.
휴대폰을 테더링하려면 USB 케이블을 사용하여 휴대폰을 컴퓨터에 연결하고 휴대폰에서 "설정 >> 연결 >> 모바일 핫스팟 및 테더링" 아래에 있는 "USB 테더링"을 눌러서 선택하세요.
테더링은 Windows®7, Windows Vista® 및 Linux에서 작동합니다. 운영 체제와 테더링에 대한 자세한 내용을 보려면 www.android.com/tether를 방문하세요.
Windows XP의 경우 휴대폰을 테더링하기 전에 컴퓨터에 USB 드라이버를 설치해야 합니다. 드라이버를 설치하려면 컴퓨터의 브라우저에서 다음 URL을 입력하세요.
http://www.samsung.com/support/
USB 케이블을 사용하여 휴대폰을 컴퓨터에 연결할 때 테더링을 통해 모바일 데이터 연결을 공유하거나 파일을 공유할 수 있습니다. 두 가지 방법을 모두 사용할 수는 없습니다. 휴대폰의 메모리 카드에 액세스하기 위해 컴퓨터를 사용하려는 경우에는 테더링을 실행하지 마세요.

모바일 핫스팟

휴대폰을 휴대용 Wi-Fi 핫스팟으로 전환하여 다른 기기와 모바일 데이터 연결을 공유할 수 있습니다.
휴대폰의 핫스팟 기능을 사용하려면 사용하는 요금제에 스마트폰 모바일 핫스팟 서비스를 추가해야 합니다.
스마트폰 모바일 핫스팟을 실행하려면 "설정 >> 연결 >> 모바일 핫스팟 및 테더링" 아래에 있는 "모바일 핫스팟"을 눌러서 선택하세요.
스마트폰 모바일 핫스팟을 설정하려면 "모바일 핫스팟"을 누른 후 "모바일 핫스팟 설정"을 누르세요.
"모바일 핫스팟 설정 >> 고급 옵션 표시 >> 최대 연결" 아래에서 모바일 데이터 연결을 사용할 수 있는 기기 수를 설정할 수 있습니다.
