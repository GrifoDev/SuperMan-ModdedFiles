Modem USB

Utilisez le modem USB pour partager la connexion au réseau mobile de votre appareil avec un PC via un câble USB. Pour activer le modem USB, le service Smartphone Mobile Hotspot doit être compris dans votre forfait.
Pour connecter votre appareil, reliez-le à un ordinateur à l'aide d'un câble USB. Sur votre appareil mobile, accédez ensuite à Paramètres > Connexions > Point d\'accès mobile et modem, puis sélectionnez modem USB.
Le modem USB fonctionne avec Windows et Linux.
Pour plus d'informations, notamment concernant l'utilisation du modem USB avec d'autres systèmes d'exploitation, rendez-vous sur http://www.android.com/tether.
Pour Windows XP, vous devrez peut-être installer un pilote USB sur votre PC avant afin de pouvoir connecter votre appareil. Pour installer les pilotes, rendez-vous sur http://www.samsung.com/support/.
Lorsque vous connectez votre appareil à un PC à l'aide d'un câble USB, vous avez le choix entre partager votre connexion au réseau mobile via le modem ou partager des fichiers. Vous ne pouvez pas faire les deux. N'activez pas le modem USB si vous souhaitez utiliser votre PC pour accéder à la carte SD de votre appareil.

Point d'accès mobile

Convertissez votre appareil en point d'accès Mobile Hotspot pour partager votre connexion des données mobiles avec d'autres appareils.
Pour activer Mobile Hotspot, le service Smartphone Mobile Hotspot doit être compris dans votre forfait.
Pour activer Smartphone Mobile Hotspot, accédez à Paramètres > Connexions > Point d\'accès mobile et modem, puis sélectionnez Mobile Hotspot.
Pour configurer Mobile Hotspot pour smartphones, appuyez sur Mobile Hotspot, puis sur Configurer point d\'accès mobile.
Pour définir le nombre d'appareils qui peuvent utiliser votre connexion au réseau mobile, accédez à Configurer point d\'accès mobile > Afficher les options avancées > Connexions maximales.
