Anclaje de USB

Usa el modem para compartir la conexión de datos de tu teléfono con una computadora por medio de un cable USB.
Para vincular tu teléfono conéctalo a una computadora con el cable USB y toca para seleccionar “Modem USB” ubicado en Ajustes >> Modem y Mobile HotSpot. Modem funciona con Windows®7, Windows Vista®, y Linux. Para más información sobre modem con sistemas operativos, visitahttp://www.android.com/tether.
