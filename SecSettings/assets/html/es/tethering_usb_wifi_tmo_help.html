Anclaje a red USB

Use el anclaje a red USB para compartir la conexión de red móvil del dispositivo con una computadora usando un cable USB. Para activar un anclaje a red USB, debe tener un servicio Smartphone Mobile Hotspot incluido en su plan de pago.
Para anclar a red su dispositivo, conéctelo a una computadora a través de un cable USB. Luego, en su dispositivo móvil vaya a “Ajustes > Conexiones > Anclaje a red y Mobile Hotspot, y seleccione anclaje a red por USB.
El anclaje USB funciona con Windows y Linux
Para obtener más información, incluido cómo usar el anclaje a red de USB con otros sistemas operativos, visite http://www.android.com/tether.
Para Windows XP, puede que sea necesario que instale un controlador USB en su computadora antes de poder anclar a red su dispositivo. Para instalar los controladores, vaya a http://www.samsung.com/support/.
Cuando conecta su dispositivo a una computadora con un cable USB, puede compartir su conexión de red móvil al usar el anclaje a red o puede compartir archivos. No puede hacer ambas acciones al mismo tiempo. No active el anclaje a red por USB si desea usar su computadora para acceder a la tarjeta de memoria de su dispositivo.

Mobile Hotspot

Cambie su dispositivo a Mobile Hotspot para compartir una conexión de datos móviles con otros dispositivos.
Para activar Mobile Hotspot, debe tener un servicio de Smartphone Mobile Hotspot incluido en su plan de pago.
Para activar Smartphone Mobile Hotspot, vaya a “Ajustes > Conexiones > Anclaje a red y Mobile Hotspot, y seleccione Mobile Hotspot.
Para configurar Mobile Hotspot de su smartphone, pulse Mobile Hotspot y luego Configurar Mobile Hotspot.
Para configurar cuántos dispositivos pueden usar su conexión de red móvil, vaya a Configurar Mobile Hotspot > Mostrar opciones avanzadas > Máximo de conexiones.
