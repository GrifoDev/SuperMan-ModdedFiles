USB tethering

Usa la inmovilización para compartir la conexión móvil de datos tu teléfono con una sola computadora mediante un cable USB.
Para vincular tu teléfono a una computadora con u cable USB en tu teléfono toca para seleccionar “Inmovilización USB” localizado en “Ajustes >> Conexiones >> Anclaje a red y Zona portátil”.
Inmovilización funciona con Windows®7, Windows Vista®, y Linux. Para más información sobre inmovilización con sistemas operativos, visita www.android.com/tether.
Para Windows XP, es posible que tengas que instalar un USB driver en tu computadora antes de que puedas vincular tu teléfono. Para instalar los drivers introduce el siguiente URL en el navegador de tu computadora.
<http://www.samsung.com/support/>
Cuándo conectas tu teléfono a una computadora con un cable USB, puedes compartir tu conexión de datos móviles al inmovilizar O puedes compartir archivos—no puedes hacer ambas. No actives inmovilización si quieres usar tu computadora para tener acceso a la tarjeta de memoria de tu computadora.

Zona portátil

Convierte tu teléfono en un Wi-Fi Hotspot portátil, podrás compartir t conexión de datos móviles con otros dispositivos.
Para usar las capacidades de Hotspot de tu teléfono, debes tener el servicio de Smartphone Mobile Hotspot agregado a tu plan de servicios.
Para activar Smartphone Mobile Hotspot toca para seleccionar “Zona portátil” localizado bajo “Ajustes >> Conexiones >> Anclaje a red y Zona portátil”.
Para configurar Smartphone Mobile Hotspot, toca en “Zona portátil”, luego en “Configurar Zona portátil”.
Puedes establecer cuantos dispositivos puedes usar con tu conexión de dato móviles bajo “Configurar Zona portátil >> Mostrar opciones avanzadas >> Máximo de conexiones”.
