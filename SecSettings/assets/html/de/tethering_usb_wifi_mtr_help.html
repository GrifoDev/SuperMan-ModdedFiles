USB tethering

Mit Tethering geben Sie die mobile Datenverbindung Ihres Telefons über ein USB-Kabel für einen einzelnen Computer frei.
Zum Tethering verbinden Sie Ihr Telefon über ein USB-Kabel mit einem Computer, und tippen Sie auf Ihrem Telefon unter “Einstellungen >> Verbindungen >> Mobile Hotspot und Tethering” auf “USB-Tethering”.
Tethering funktioniert mit Windows®7, Windows Vista® und Linux. Weitere Informationen zum Tethering mit Betriebssystemen finden Sie unter www.android.com/tether.
Bei Windows XP muss ggf. ein USB-Treiber auf dem Computer installiert werden, bevor Sie das Tethering mit Ihrem Telefon vornehmen können. Zum Installieren der Treiber geben Sie die nachstehende URL im Browser Ihres Computers ein.
<http://www.samsung.com/support/>
Wenn Sie Ihr Telefon über ein USB-Kabel mit einem Computer verbinden, können Sie entweder die mobile Datenverbindung per Tethering freigeben ODER Daten freigeben, nicht jedoch beides. Aktivieren Sie Tethering nicht, wenn Sie mit dem Computer auf die Speicherkarte Ihres Telefons zugreifen möchten.

Mobile Hotspot

Aktivieren Sie Ihr Gerät als mobilen WLAN-Hotspot und teilen Sie Ihre mobile Datenverbindung mit anderen Geräten.
Smartphone Mobile Hotspot aktivieren Sie, indem Sie unter “Einstellungen >> Verbindungen >> Mobile Hotspot und Tethering” auf “Mobile Hotspot” tippen.
Tippen Sie zum Konfigurieren von Smartphone Mobile Hotspot auf “Mobile Hotspot” und dann auf “Mobile Hotspot konfigurieren”.
Wie viele Geräte Ihre mobile Datenverbindung nutzen können legen Sie unter “Mobile Hotspot konfigurieren >> Erweiterte Optionen anzeigen >> Maximale Anzahl an Verbindungen” fest.
