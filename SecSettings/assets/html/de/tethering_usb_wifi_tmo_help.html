USB-Tethering

Nutzen Sie USB-Tethering, sodass Sie auch mit einem per USB-Kabel verbundenen PC von der Verbindung zum Mobilfunknetz profitieren können. Um das USB-Tethering aktivieren zu können, muss Ihr Tarif den Dienst “Smartphone Mobile Hotspot“ umfassen.
Verbinden Sie Ihren PC zur Nutzung des USB-Tethering per USB-Kabel mit Ihrem Gerät. Wählen Sie anschließend auf dem Mobilgerät unter „Einstellungen > Verbindungen > Mobile Hotspot und Tethering“ die Option „USB-Tethering“ aus.
USB-Tethering funktioniert unter Windows und Linux.
Weitere Informationen, einschließlich Informationen zur Verwendung von USB-Tethering mit anderen Betriebssystemen, finden Sie unter http://www.android.com/tether.
Unter Windows XP muss ggf. ein USB-Treiber auf dem PC installiert werden, bevor Sie das Tethering mit Ihrem Gerät vornehmen können. Sie können die Treiber unter http://www.samsung.com/support/ installieren.
Wenn Sie das Gerät per USB-Kabel an einen PC anschließen, können Sie entweder die Mobilfunkverbindung per Tethering für den PC nutzen, oder Sie können Dateien übertragen. Beides gleichzeitig ist nicht möglich. Aktivieren Sie nicht das USB-Tethering, wenn Sie mit dem PC auf die SD-Karte des Geräts zugreifen möchten.

Mobile Hotspot

Aktivieren Sie Ihr Gerät als Mobile Hotspot und teilen Sie Ihre mobile Datenverbindung mit anderen Geräten.
Um Mobile Hotspot einzuschalten, müssen Sie den Dienst Smartphone Mobile Hotspot in Ihren Vertrag aufnehmen lassen.
Sie können Smartphone Mobile Hotspot aktivieren, indem Sie zu „Einstellungen > Verbindungen > Mobile Hotspot und Tethering” gehen und „Mobile Hotspot” auswählen.
Tippen Sie zum Konfigurieren von Smartphone Mobile Hotspot auf „Mobile Hotspot“ und dann auf „Mobile Hotspot konfigurieren“.
Unter „Mobile Hotspot konfigurieren > Erweiterte Optionen anzeigen > Maximale Anzahl an Verbindungen“ können Sie festlegen, wie viele Geräte die Mobilfunkverbindung nutzen dürfen.
